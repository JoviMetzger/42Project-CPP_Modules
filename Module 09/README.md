## ðŸŒ€ Exercise 02
### Description
This program sorts a positive integer sequence with the Ford-Johnson algorithm. <br>
**Container: vector & list** <br>
<br>
### Installation
1. Compile the program:
```bash 
$ make 
```
2. Run the program:
```bash 
$  ./PmergeMe 3 5 9 7 4
```
OR
```bash 
$  ./PmergeMe `shuf -i 1-100000 -n 3000 | tr "\n" " "`
```
